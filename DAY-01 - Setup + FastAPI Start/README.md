## ▶️ How to Run

### 1. Activate the virtual environment
```bash
# From the root /AI folder
source venv/bin/activate
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Start the server
```bash
uvicorn main:app --reload
```

### 4. Open in browser
```
http://127.0.0.1:8000/test
http://127.0.0.1:8000/hello
```

> `--reload` means the server **automatically restarts** whenever you save changes to your code. Super useful while learning!

---

## 💡 Key Concepts Learned

- **FastAPI** uses Python decorators like `@app.get("/route")` to define API endpoints
- A **route** is just a URL path your API responds to
- **Uvicorn** is the server that actually runs FastAPI
- **Virtual environment** (`venv`) isolates your project's packages from the rest of your system

---

## 🗒️ Notes

- `__pycache__/` is auto-generated by Python — it caches compiled code to make reruns faster. You can ignore it.
- Always activate `venv` before running or installing anything!
